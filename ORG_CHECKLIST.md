# GroundZero Community Setup Checklist

Use this checklist to set up the organization on GitHub.

## 1. Publish Organization Profile README

- Create or use repository named `.github` under your organization.
- Add file `profile/README.md` using the prepared content from this workspace.
- Commit to `main` branch.
- Confirm the README appears on organization Overview.

## 2. Clean Organization Profile Metadata

In Organization Settings -> Profile:

- Name: Ground Zero Community
- Short bio: A community for builders in software, design, AI, and cybersecurity.
- Location: Philippines
- Website: https://groundzero.tech
- Public email: info@groundzero.tech
- X/Twitter: optional (only if active)

## 3. Fix and Reduce Social Links

Keep only intentional links (avoid duplicates):

- Facebook Page: https://www.facebook.com/GroundZeroDigital/
- Facebook Group: https://www.facebook.com/groups/groundzerocommunity/
- Discord: https://discord.gg/4H2v6UwjmY

## 4. Pin Repositories Strategically

Select up to 6 pinned repositories that represent quality and direction:

- 1 flagship project
- 1 beginner-friendly project
- 1 documentation/resources repository
- 1 tools/utilities repository
- 1 design or frontend showcase
- 1 AI or cybersecurity project

## 5. Standardize Repository Basics

For every active repository:

- Add clear README with setup steps
- Add LICENSE
- Add CONTRIBUTING.md
- Add CODE_OF_CONDUCT.md
- Add issue templates and pull request template
- Add branch protection on `main`
- Enable Dependabot security updates

## 6. Establish Community Contribution Flow

- Define labels: `bug`, `enhancement`, `good first issue`, `help wanted`, `documentation`
- Add project board for roadmap and triage
- Enforce PR review before merge
- Require status checks for CI

## 7. Community Operations

- Publish a monthly roadmap/update post
- Keep a `resources` or `awesome` list updated
- Highlight top contributors and community projects
- Maintain response SLA for questions/issues

## 8. Immediate Quick Wins (Today)

- Publish organization profile README
- Remove duplicate links in org profile
- Correct public email format to `info@groundzero.tech`
- Pin high-quality repositories
- Enable branch protections for important repos
