# .github Repository

This repository contains organization-wide community health files, default templates, and configuration for the [GroundZero Community](https://github.com/GroundZero-Community).

## Repository Contents

- **[profile/README.md](profile/README.md)**: The profile README displayed on the GroundZero Community organization landing page.
- **[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)**: The organization-wide Code of Conduct.
- **[CONTRIBUTING.md](CONTRIBUTING.md)**: Guidelines for contributing to GroundZero projects.
- **[SECURITY.md](SECURITY.md)**: The default security policy for our repositories.
- **[LICENSE](LICENSE)**: The default MIT License for our community files.
- **[.github/ISSUE_TEMPLATE/](.github/ISSUE_TEMPLATE/)**: Organization-wide issue templates for bug reports and feature requests.
- **[.github/pull_request_template.md](.github/pull_request_template.md)**: Default template for pull requests.
- **[.github/FUNDING.yml](.github/FUNDING.yml)**: Default sponsor settings.

## How it works

GitHub automatically applies these files as organization-wide defaults for any repository under `GroundZero-Community` that does not define its own community health files.
